## 🎯 补丁功能说明

### 生成补丁
1. 点击"生成问号补丁"按钮
2. 自动从数据库读取自定义物品（entry >= 910000）
3. 导出CSV并合并到基础Item.dbc
4. 生成的补丁保存在 `~/WOWItemMaker_Patch/Item.dbc`

### 推送补丁
1. 点击"推送补丁到服务器"按钮
2. 自动识别当前环境（测试服/正式服）
3. SCP上传到服务器对应目录
4. 推送路径：
   - 测试服：/www/wwwroot/wow/patches/test/
   - 正式服：/www/wwwroot/wow/patches/release/
